# Gestern-Support
Support resources, FAQ, and contact information for the Gestern iOS app.

Thanks for using Gestern!

## Contact
For help, feedback, or bug reports, email: **scott.carper@icloud.com**

## FAQ

### Weather isn’t available
Gestern uses your current location to fetch weather data.  
If weather is unavailable, check:
- iOS Settings → Privacy & Security → Location Services → Gestern → **While Using**

### News / History not loading
If you’re offline or on a restricted network, data may not load. Try again on a stable connection.

## Privacy
Gestern uses location only to provide local weather while the app is in use.  
Gestern does not require an account and does not sell personal data.
